## Overview

CashGrow is a modern financial application designed to help users manage and grow their finances. I serve as the Full-Stack Engineer, responsible for both frontend and backend development.

## Technical Implementation

### Frontend Architecture
- Built responsive UI with React and TypeScript
- Implemented state management for complex financial data
- Created intuitive data visualizations for financial insights

### Backend Development
- Designed RESTful API with Node.js
- Implemented PostgreSQL as the OLTP database
- Built secure authentication and authorization systems

### DevOps & Deployment
- Containerized application using Docker
- Deployed backend on Google Cloud Platform
- Deployed frontend on Netlify with CI/CD pipeline
