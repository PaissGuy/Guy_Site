## Overview

At Israel Aerospace Industries, I led the development of advanced computer vision systems for defense applications. This role involved close collaboration with customers to deliver proof-of-concept implementations of cutting-edge algorithms.

## Technical Achievements

### Algorithm Development
- Designed and implemented NUC algorithms for sensor calibration
- Built denoising pipelines for improved image quality
- Developed motion correction using optical flow techniques

### Object Tracking
- Implemented RANSAC-based feature matching
- Designed Kalman filter tracking systems
- Achieved robust tracking under challenging conditions

### Research & Prototyping
- Conducted research on object detection and recognition
- Developed GUI tools in MATLAB and Python for testing
- Performed data analysis with MySQL on large datasets

### Documentation & Collaboration
- Documented algorithms in MS Office, Word, and LaTeX
- Worked directly with customers for POC demonstrations
- Utilized SVN for version control
